# 📄 ResumeIQ – AI Resume Analyzer

🚀 A smart resume analysis tool that evaluates resumes based on job roles using keyword matching and provides actionable feedback.

---

## 🔥 Features

- 📊 ATS Score Calculation  
- 🎯 Role-Based Analysis (QA, Developer, Data Scientist, etc.)  
- ❌ Missing Keywords Detection  
- 💡 Smart Suggestions  
- 🎨 Premium UI (Glassmorphism + Animations)  
- 📄 Resume Text Extraction from PDF  

---

## 🛠 Tech Stack

- Python  
- Streamlit  
- pdfplumber
- Plotly

##🧠 How It Works

1. Upload your resume (PDF)
2. Select your target job role
3. System analyzes resume using:
   Keyword matching
   Skill comparison
4. Outputs:
   ATS Score
   Missing Skills
   Visual Dashboard

---

##📊 Example Output

ATS Score: 80%
Matched Skills: 4
Missing Skills: SQL, API
